# Bioprotocol
This repository contains FIJI scripts (.ijm and .py), as well as CellProfiler pipelines (.cppipe) from the protocol paper ### (DOI: ###)

# Installation
**FIJI Scripts:** Copy the folder *Semi-Automated Analysis scripts* to the **Plugins** folder in your FIJI directory and restart Fiji. Once inside FIJI, you can find the folder with the scripts in the **Plugins** tab.

**CellProfiler Pipeline:** Drag and drop the desired pipeline onto the empty space on the left side

# Software versions
**FIJI/ImageJ:** 1.54p  
**CellProfiler:** 4.2.8
